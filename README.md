# Пробуем запускать сервер nginx из под docker

# Сборка и запуск сервера 

Запускаем сервер по команде в терминале:

```
docker-compose up -d
```

# Работа

Открываем в браузере по ссылке http://localhost:8000/

# Выключение

Останавливаем сервер
```
docker-compose down
```

Удаляем образ сервера
```
docker rmi nginx
```
